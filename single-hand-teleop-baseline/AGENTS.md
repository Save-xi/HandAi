# AGENTS 说明

1. 本项目当前阶段只聚焦单右手 baseline。
2. 不得默认扩展到双手或多手主流程。
3. 不得把 Unity 运行时当成 baseline 的前置依赖。
4. 不得把真机 SVH 控制、串口服务或网络链路当成 baseline 启动前提。
5. 修改代码前优先保持模块边界清晰，避免把可选扩展耦合进主链路。
6. 新增能力时优先复用现有 JSON / JSONL 输出接口和既有 contract。
7. 如果需求不够明确，优先保持简单、可运行、可验证。
8. 默认将仓库内新增或修改的注释、文档、日志文案、CLI 帮助、工作流名称、测试说明等文字类内容写成中文。
9. 如果字段名、schema 键、协议常量、外部接口、依赖名称等必须保留英文，不得强行翻译；必要时在相邻注释中用中文解释。
